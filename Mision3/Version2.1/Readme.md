Proyecto Chatbotv3:  Mision3 crear un chatbot con IA groq y las librerias de python
La diferencia con la version 1: El chatbot supervisado chatbot.sp.py selecciona una respuesta de una tupla definida basada en probabilidades. con groq genera respuestas por cada pregunta.
Groq ya esta prentrenado por lo tanto no es necesario funciones de entrenamiento
Groq entiende el contenido semantico y contexto, sin necesidad de vectores manuales

.Instalacion de groq desde el entorno virtual env3.13.7: pip install groq
.Verificar instalacion de groq y version groq 1.0.0
